## ✨ 개요(Intro)

**엘리스 SW 엔지니어 트랙 9기** 2차 프로젝트 과제로서,

실제 서비스 운영이 아닌 기능 구현에 초점을 맞춘 데모 웹 어플리케이션입니다.

모바일 웹페이지를 기준으로 두고 구성하였고 React, Style-components를 주요 스택으로 사용하였습니다.

Stay and go는 소비자가 다양한 숙소를 쉽게 찾고 예할 수 있는 온라인 플랫폼입니다.
